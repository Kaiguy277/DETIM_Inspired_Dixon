## 1. Geodetic Mass Balance (Hugonnet et al. 2021)

**File:** `dixon_glacier_hugonnet.csv`

**Source:** Hugonnet, R. et al. (2021). Accelerated global glacier mass loss in the
early twenty-first century. *Nature*, 592, 726–731.
https://doi.org/10.1038/s41586-021-03436-z

**Method:** ASTER L1A stereo satellite imagery (15 m resolution) processed into DEMs.
Elevation change time series generated by stacking many overlapping ASTER DEM pairs
per glacier and fitting temporal trends per pixel. This is a completely independent
dataset from any WorldView/ArcticDEM or SfM products.

**Data for Dixon Glacier:**

| Period | dh/dt (m/yr) | ± err | MB (m w.e./yr) | ± err | Area measured |
|--------|-------------|-------|----------------|-------|---------------|
| 2000–2010 | -1.261 | 0.246 | -1.072 | 0.225 | 99.5% |
| 2010–2020 | -0.948 | 0.226 | -0.806 | 0.202 | 99.5% |
| 2000–2020 | -1.105 | 0.115 | -0.939 | 0.122 | 99.5% |

Thinning decelerated slightly from 2000–2010 to 2010–2020.

### CSV column definitions

| Column | Units | Description |
|--------|-------|-------------|
| `rgiid` | -- | RGI v6 glacier identifier |
| `period` | date_date | Start and end dates (YYYY-MM-DD_YYYY-MM-DD) |
| `area` | m² | Glacier area used in the calculation |
| `dhdt` | m/yr | Mean glacier-wide surface elevation change rate |
| `err_dhdt` | m/yr | 1-sigma uncertainty on dh/dt |
| `dvoldt` | m³/yr | Volume change rate (= dhdt × area) |
| `err_dvoldt` | m³/yr | Uncertainty on volume change rate |
| `dmdt` | Gt/yr | Mass change rate (volume × density 850 kg/m³) |
| `err_dmdt` | Gt/yr | Uncertainty on mass change rate |
| `dmdtda` | m w.e./yr | **Specific mass balance** (mass change per unit area) |
| `err_dmdtda` | m w.e./yr | Uncertainty on specific mass balance |
| `perc_area_meas` | fraction | Fraction of glacier area with valid measurements |
| `perc_area_res` | fraction | Fraction of glacier area with interpolated results |
| `valid_obs` | count | Total number of valid elevation observations |
| `valid_obs_py` | count/yr | Valid observations per year |
| `reg` | -- | RGI first-order region number (1 = Alaska) |

### How to use for model calibration

The key field for calibration is **`dmdtda`** (specific mass balance in m w.e./yr).
This is the glacier-wide average mass change rate that a model should reproduce.
